# python-password-generator
A simple Python script to generate secure, random passwords using letters, numbers, and special characters.

Features
- Choose password length.
- Includes security against incorrect input.
- Uses secure and random characters from string library.

Installation

Requirements
- Python 3.x installed.

Steps
1. Download main.py
2. Open your terminal or command prompt
3. Run the script with python main.py

Usage

When the script starts, it asks for a number:

How long must the password be? 15

The output is then, for example:

Result: !7tG2y]p-9fo2

How it works

- random: for choosing random characters.
- string: to quickly retrieve a list of letters and symbols.
- try/expect: To prevent the program from crashing on incorrect input.
